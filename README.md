# 项目背景

本项目为学习`ThreadX`及其相关组件提供了一个硬件平台以及相关代码示例。

本仓库所包含文件夹如下：

* Hardware: 硬件相关
* Software: 软件代码

# 安装

克隆本项目到本地

```shell
git clone https://github.com/HamsterAPig/ThreadXBoard.git
```

如果你不希望参与本项目的开发或者因下载速度过慢，你也可以选择仅克隆当前版本

```shell
git clone https://github.com/HamsterAPig/ThreadXBoard.git --depth=1
```
